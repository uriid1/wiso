# Install
1) git clone https://github.com/uriid1/wiso/
2) cd wiso_1.0
3) chmod +x install
4) sudo ./install
5) Done!

# Using
Open a terminal and run wiso. After that, just follow the instructions.

# Uninstall
sudo rm /usr/bin/wiso
